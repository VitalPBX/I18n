iax_settings.maxcallnumbers = Números máximos de chamadas
iax_settings.maxcallnumbers.tooltip = A opção 'maxcallnumbers' limita a quantidade de números de chamadas permitidos para cada endereço IP remoto individual. Quando um endereço IP atinge seu limite de número de chamadas, não são permitidas novas conexões até que as anteriores sejam fechadas. Essa opção também pode ser usada em uma definição de mesmo nível, mas só entra em vigor para o IP de um ponto dinâmico depois de concluir o registro.
iax_settings.maxcallnumbers_nonvalidated = Número máximo de chamadas não validadas
iax_settings.maxcallnumbers_nonvalidated.tooltip = O 'maxcallnumbers_nonvalidated' é usado para definir o número combinado de números de chamada que podem ser alocados para conexões em que a validação do token de chamada foi desativada. Ao contrário da opção "maxcallnumbers", esse limite não é separado para cada endereço IP individual. Qualquer conexão que resulte em um número de chamada validado de token de não chamada sendo alocado contribui para esse limite.
