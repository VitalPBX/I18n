conferences.announce_join_leave = Анонсировать участника
conferences.announce_join_leave.tooltip = При входе в конференцию пользователю будет предложено записать свое имя . После того, как имя будет записано, оно будет воспроизводиться, когда пользователь входит и выходит из конференции.
conferences.announce_only_user = Пустая конференция
conferences.announce_only_user.tooltip = Сообщить пользователю о входе в пустую конференцию.
conferences.announce_user_count = Кол-во участников
conferences.announce_user_count.tooltip = Сообщить кол-во участников новому пользователю конференции.
conferences.announce_user_count_all = Считать участников
conferences.announce_user_count_all.tooltip = Используется для объявления количества участников всем в конференции. Если это значение является числом, то сообщение воспроизводится только тогда, когда количество участников превышает установленное число. Доступны следующие варианты: <b>yes</b>, <b>no</b> или <b>число</b>. По умолчанию - <b>no</b>.
conferences.description = Описание
conferences.description.tooltip = Краткое описание для идентификации этой конференции.
conferences.dsp_drop_silence = Удалять тишину
conferences.dsp_drop_silence.tooltip = Данная опция отбрасывает голосовые пакеты активного участника, в которых не распознается речь, т.е. тишину. Включение этой опции кардинально уменьшает нагрузку на сервер и улучшает качество звука в конференции, за счет удаления фоновых шумов. Рекомендуется использовать для больших конференций со множеством участников.
conferences.end_marked = Исключать всех участников
conferences.end_marked.tooltip = Исключать всех участников из конференции (закончить конференцию), когда ее покинет последний из организаторов.
conferences.extension = Номер
conferences.extension.invalid = Недопустимый номер конференции. Разрешены только символы *, #, + и цифры.
conferences.extension.tooltip = Внутренний номер, набираемый участником, для доступа к данной конференции.

conferences.max_members = Макс. число участников
conferences.max_members.no_limit = Без ограничений
conferences.max_members.tooltip = Данный параметр устанавливает лимит участников конференции. Когда предел достигнут, конференция будет заблокирована до тех пор, пока кто-то не уйдет. Пользователь с правами администратора, может подключаться независимо от состояния лимита.
conferences.music_group_id = Музыка на удержании
conferences.music_group_id.tooltip = Данный параметр определяет, какая музыка будет использоваться, когда участник находится в режиме ожидания.
conferences.music_on_hold_when_empty = Музыка для одного
conferences.music_on_hold_when_empty.tooltip = Данный параметр определяет, должна ли воспроизводиться музыка участнику, ожидающему организатора конференции, или когда в конференции присутствует только один участник.
conferences.quiet = Тихий режим
conferences.quiet.tooltip = Если включено, то не проигрывать сообщения о входе/выходе из конференции других участников. Не влияет на такие сообщения, как, например, ввод пин-кода.
conferences.record_conference = Запись
conferences.record_conference.tooltip = Запись конференции ведется с момента, когда первый пользователь входит в конференцию, и заканчивается, когда последний пользователь покидает конференцию.
conferences.settings = Настройки
conferences.startmuted = Начать без звука
conferences.startmuted.tooltip = Установить, если все участники должны начать конференцию в тишине (временно отключается микрофон).
conferences.talk_detection_events = Распознавание речи
conferences.talk_detection_events.tooltip = Этот параметр задает, должны ли отправляться уведомления в виде событий через Asterisk Manager Interface (AMI) о том, что пользователь начал и закончил говорить.
conferences.userpin = PIN пользователя
conferences.userpin.invalid = PIN пользователя недействителен. Используйте только цифры и символ «*» (звездочка).
conferences.userpin.tooltip = Устанавливает, должны ли пользователи вводить PIN-код до входа в конференцию.
conferences.video_mode = Видео режим
conferences.video_mode.first_marked = Админ
conferences.video_mode.first_marked.tooltip = Первый администратор, присоединяющийся к конференции с видеовозможностью, является единственным источником распространения видео для всех участников. Если администратор уходит, следующий администратор, присоединившийся после них, становится источником.
conferences.video_mode.follow_talker = Следить за говорящим
conferences.video_mode.follow_talker.tooltip = Видео передается от участника который в данный момент говорит (если он его транслирует).
conferences.video_mode.none = Нет
conferences.video_mode.none.tooltip = По умолчанию в конференции видео не транслируется. В любое время пользователь по-прежнему может быть установлен в качестве источника видео с помощью AMI или DTMF действия.
conferences.wait_marked = Ждать организатора
conferences.wait_marked.tooltip = Устанавливает, должны ли пользователи ждать, пока организатор не войдет в конференцию.
